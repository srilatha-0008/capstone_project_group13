# Uber Trip Fare Prediction Pipeline

## Project Overview

This project builds a complete end-to-end Data Engineering and Machine Learning pipeline using:

- AWS S3
- Databricks
- PySpark
- ETL Pipeline
- Feature Engineering
- Machine Learning
- dbt
- Cassandra DB
- FastAPI
- Docker
- Kubernetes
- Tableau
- GitHub
